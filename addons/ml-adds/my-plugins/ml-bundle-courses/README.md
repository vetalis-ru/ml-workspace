# ML Bundle Courses

ML Bundle Courses — плагин для пошаговой выдачи уровней доступа (УД) по цепочкам «Сборных курсов» в экосистеме MemberLux.

Идея: студент проходит курсы последовательно. После выдачи сертификата за текущий курс автоматически назначается следующий УД в цепочке, с индивидуальным сроком (duration/units).

## Основные возможности
- Пошаговая выдача УД на событие выдачи сертификата (hook `mbl_certificate_issued`)
- Программы (сборные курсы) как CPT `ml_program`
- Шаги программы хранятся в `post_meta` (`mlp_steps`) в виде массива `{term_id, duration, units}`
- Привязка пользователя к программе хранится в `user_meta`
- Уведомления и сохранение прогресса после выдачи следующего шага
- Идемпотентность: повторное событие не должно повторно выдавать следующий шаг
