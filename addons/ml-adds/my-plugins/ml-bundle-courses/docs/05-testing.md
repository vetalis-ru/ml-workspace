# План тестирования (v0.2.0)

## Подготовка
1) Создать программу (ml_program) и заполнить шаги
2) Привязать тестового пользователя к программе
3) Убедиться, что у пользователя выдан УД первого шага

## Тест-кейсы

### Кейс 1: выдача сертификата по шагу 1
Ожидается:
- выдаётся УД следующего шага
- прогресс обновляется

### Кейс 2: выдача сертификата по последнему шагу
Ожидается:
- ничего нового не выдаётся

### Кейс 3: повторное событие
Ожидается:
- следующий шаг не выдаётся повторно (идемпотентность)

### Кейс 4: пользователь не в программе
Ожидается:
- событие игнорируется, выдачи нет
