# Инструкция администратора (v0.2.0)

## 1. Программы / Сборные курсы
В админке WordPress доступен тип записи **«Сборные курсы»** (`ml_program`).

Программа состоит из шагов:
- Уровень доступа (term_id / УД)
- Длительность (duration)
- Единицы (units: days или months)

Шаги упорядочены: порядок критичен.

## 2. Управление участниками
Плагин предоставляет страницы управления:
- **«Связать пользователя»** — поиск пользователя (email/логин/имя/фамилия/ID) и привязка к программе.
- **«Участники программ»** — список всех привязанных пользователей с фильтрами по программе/шагу и сортировкой по столбцам.

## 3. Что происходит при выдаче сертификата
Когда пользователю выдан сертификат за текущий курс:
- плагин определяет текущий шаг программы
- если есть следующий шаг — выдаёт следующий УД с заданным сроком
- сохраняет прогресс пользователя
