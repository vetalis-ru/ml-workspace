# Обзор (v0.2.0)

ML Bundle Courses 0.2.0 — плагин для пошаговой выдачи уровней доступа (УД) по цепочкам «Сборных курсов».

Для пользователя процесс прозрачен: после выдачи сертификата за текущий курс автоматически назначается следующий УД в цепочке, с индивидуальным сроком и единицами времени.

Логика запускается на хук `mbl_certificate_issued`: извлекается сертификат, определяется текущий `term_id`, выбирается следующий шаг, создаётся ключ доступа и активируется через стандартные функции MemberLux, после чего отправляются уведомления и сохраняется прогресс пользователя.
